## Завдання 1. Базові команди

### 1\. Показати вміст домашнього каталогу

``` bash
ls \\\~
```

### 2\. Перейти у каталог Downloads і показати його вміст

``` bash
cd \\\~/Downloads
ls
```

### 3\. Створити порожній файл (без touch)

``` bash
echo "" > file.txt
```

### 4\. Переглянути вміст файлу

``` bash
cat file.txt
```

### 5\. Перейти у домашній каталог абсолютним шляхом

``` bash
cd /home/username
```

### 6\. Перейти у домашній каталог відносним шляхом

``` bash
cd \\\~
```

## Завдання 2.

``` bash
man ls
help cd
man cat
man man
cp --help
mv --help
```

### Питання та відповіді

**1. Який ключ ls показує приховані файли?**

Ключ `-a` показує всі файли, включаючи приховані.

Приклад:

``` bash
ls -a
```

**2. Який ключ у cat нумерує рядки?**

Ключ `-n` нумерує всі рядки у файлі.

Приклад:

``` bash
cat -n file.txt
```

**3. Чим відрізняються man і --help?**

`man` відкриває повну документацію команди з детальним описом та всіма
параметрами.

`--help` показує коротку довідку по використанню команди прямо в
терміналі.
